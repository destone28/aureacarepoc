# Assets

Cartella per asset binari (loghi, immagini sponsor).

## File attesi

- `3dsprinted-logo.png` — logo 3DSprinted usato nella sezione sponsor di `home.html` e `profile.html`.
  Se l'immagine non viene caricata, il markup mostra automaticamente un fallback SVG inline
  con glifo stilizzato + testo "3DSprinted".

Per sostituire/aggiungere il logo reale: copia il file PNG (sfondo bianco/trasparente, altezza ~60-80px)
in questa cartella con il nome esatto `3dsprinted-logo.png`.
